A fun and interactive full-stack web application where users guess famous destinations based on cryptic clues! Challenge your friends and see who can get the highest score.


---

## Tech Stack  

### **Frontend**  
- **React.js** (Vite) - Fast and modern UI development  
- **Material UI (MUI)** - Prebuilt UI components  
- **Tailwind CSS** - Utility-first styling  
- **Framer Motion** - Smooth animations  
- **Redux Toolkit** - State management  

### **Backend**  
- **Node.js** - JavaScript runtime  
- **Express.js** - Backend framework  
- **MongoDB** - NoSQL database  
- **Mongoose** - MongoDB ORM  

### **Tools & Services**  
- **Thunder Client** - API testing  
- **Git & GitHub** - Version control  
- **VS Code** - Code editor  

---

## Features  
Play a destination guessing game with cryptic clues  
Challenge your friends and compare scores  
Animated UI with smooth transitions  
Real-time feedback on answers  
Secure and scalable backend with MongoDB  
Responsive design for mobile & desktop
